tutorial de uso do git e github

para usar crie um repositorio no seu github

    como instalar o git e usar no vs code

  • de um nome para esse repositorio
  • baixe o git controle de versionameto em sua maquina
  • na pasta de seu projeto dentro do vs code abra o terminal
  • se der erro reinicie o vs code
  • se o erro persistir confira a nas variaveis de ambiente
  • dentro de path na parte de baixo confira se se caminho do git esta correto

    como usar os comandos git

  • com o terminal aberto dentro do seu projeto de o comando.
  • git init
  • git add (serve para adicionar arquivo por arquivo ) exemplo git add index.html
  • para adicionar todos os arquivos use git add .
  • git commit -m "versao_001" serve para fazer um commit de todos os meus arquivos
  • o comando acima ira dar errado se nao configurar ele primeiro

    • como configurar

  • git config global user.email "sdjuniorsantos339@gmail.com" -no meu caso
  • git config --global user.name "junior santos" -no meu caso
  • depois disso ......
  • git branch -M main
  • mude para o link ssh e cole no meu caso
  • git remote add origin git@github.com:junior339/projeto-html_Puro.git
  • git push -u origin main
  • apos isso dara um erro de ssh click em yes
  • va ao seu github e configure uma nova chave ssh
  • no icone a direita superior va a configurações
  • chave ssh
  • configure com qualque nome
  • no seu computador crie uma chave ssh
  • no seu cmd pode ser do projeto mesmo

    Usando o github

  • git remote -v para ver os repositorios vinculados
  • git remote remove apelido substitua o apelido pelo nome do repositorio
  • para renomear o repositorio git remote rename origin novo-origin
  • git status para ver o status

    como trazer projetos para seu pc

  • no repositorio va em code
  • copie do http o link
  • abra o terminal do vs ou o terminal normal em alguma local de sua escolha
  • dentro do terminal digite (git clone https://github.com/junior339/projeto-html_Puro.git) o exemplo de um clone http aqui
  • dessa maneira vc tem acesso ao codigo fonte em seu pc
